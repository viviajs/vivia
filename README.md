## Introduction

vivia is a simple and fast static website generator that gives you more freedom to control everything on your website through powerful plugins.

## Quick Start

**Create project**

```bash
$ npm create vivia
```

**Preview & Hot-reload writing**

```bash
$ npm run dev
```

**Generate website**

```bash
$ npm run build
```
